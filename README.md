# 🌐 Network Traffic Monitoring Dashboard

A real-time network monitoring tool built using Flask, psutil, and Chart.js. It tracks system network usage and displays it through an interactive dashboard.

---

## 🚀 Features

- Real-time upload & download speed  
- Active network connections  
- TCP vs UDP protocol distribution  
- Top applications using network  
- Data usage history  
- Live graphs and charts  

---

## 🛠️ Tech Stack

- Python (Flask)
- psutil
- HTML, CSS, JavaScript
- Chart.js

---

## ▶️ How to Run

### Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/network-traffic-monitor.git
cd network-traffic-monitor
```

### Install dependencies
```bash
python app.py
```

### Run the project
```bash
python app.py
```

### Open in browser
```bash
(http://127.0.0.1:5000)
```

---

## 🍎 macOS Note

For full network details (TCP/UDP, apps), run:
```bash
sudo python3 app.py
```

---

## 🧠 Key Concept

This project measures **real-time network throughput**, not maximum internet speed.
